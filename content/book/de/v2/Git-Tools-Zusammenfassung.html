Zusammenfassung

Sie haben eine Reihe von fortschrittlichen Tools kennengelernt, mit denen Sie Ihre Commits und die Staging-Area präziser manipulieren können. Wenn Sie Probleme bemerken, sollten Sie in der Lage sein, leicht herauszufinden, was, wann und von wem der Commit eingebracht wurde. Wenn Sie Teilprojekte in Ihrem Projekt verwenden möchten, haben Sie gelernt, wie Sie deren Anforderungen erfüllen können. An diesem Punkt sollten Sie auf der Kommandozeile die meisten der tagtäglichen Aufgaben in Git erledigen können und Sie sollten sich dabei sicher fühlen.
