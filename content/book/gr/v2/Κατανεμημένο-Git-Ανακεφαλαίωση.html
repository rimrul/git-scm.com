Ανακεφαλαίωση

Σε αυτό το σημείο θα πρέπει να αισθανόμαστε αρκετά άνετα στο να συνεισφέρουμε σε ένα έργο στο Git καθώς και να διαχειριστούμε το δικό μας έργο ή να ενσωματώσουμε τις συνεισφορές άλλων χρηστών. Μας αξίζουν συγχαρητήρια που είμαστε πλέον αποτελεσματικοί προγραμματιστής του Git! Στο επόμενο κεφάλαιο, θα μάθουμε πώς να χρησιμοποιούμε τη μεγαλύτερη και πιο δημοφιλή υπηρεσία φιλοξενίας Git, το GitHub.
