Ανακεφαλαίωση

Σε αυτό το σημείο έχουμε πλέον μία βασική γνώση του τι είναι το Git και πώς διαφέρει από ένα συγκεντρωτικό σύστημα ελέγχου έκδοσης που ενδεχομένως έχουμε χρησιμοποιήσει. Επίσης έχουμε μια έκδοση του Git στο σύστημά μας η οποία είναι ρυθμισμένη με τα προσωπικά μας στοιχεία. Ήρθε η ώρα, λοιπόν, να μάθουμε τα βασικά πράγματα για το Git.
