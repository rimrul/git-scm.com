Εξωτερικά Συστήματα

Το Git συνοδεύεται από μερικές εντολές για ενσωμάτωση με άλλα συστήματα ελέγχου έκδοσεων.

git svn

Η εντολή git svn χρησιμοποιείται για να επικοινωνήσουμε με το σύστημα ελέγχου έκδοσεων Subversion ως πελάτες. Αυτό σημαίνει ότι μπορούμε να χρησιμοποιήσουμε το Git για να κάνουμε ανακτήσουμε από και να υποβάλλουμε σε διακομιστή Subversion.

Αυτή η εντολή καλύπτεται σε βάθος στην ενότητα Git and Subversion.

git fast import

Για άλλα συστήματα ελέγχου εκδόσεων ή για εισαγωγή από σχεδόν οποιαδήποτε μορφή, μπορούμε να χρησιμοποιήσουμε τη λειτουργία git fast import για να απεικονίσουμε γρήγορα την άλλη μορφή σε κάτι που η Git μπορεί εύκολα να καταγράψει.

Αυτή η εντολή καλύπτεται σε βάθος στην ενότητα Ένας εξατομικευμένος εισαγωγέας.
