Git στο Eclipse

Το Eclipse διαθέτει ένα plugin που ονομάζεται Egit, το οποίο παρέχει μια αρκετά ολοκληρωμένη διεπαφή στις λειτουργίες Git. Μπορούμε να το προσπελάσουμε πηγαίνοντας στο Git Perspective (Window > Open Perspective > Other… και επιλέγουμε “Git”).

Το EGit συνοδεύεται από εκτενή και ποιοτική τεκμηρίωση, την οποία μπορούμε να βρούμε αν πάμε από το μενού στο Help > Help Contents και επιλέξουμε τον κόμβο “EGit Documentation” από τη λίστα περιεχομένων.
